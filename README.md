# Projeto: Monitoramento de evolução motora em pacientes pós-AVC com sequelas em membros inferiores - Grupo 4 (FICSAE, 4º Semestre)

## Descrição do Projeto
Desenvolvimento de uma tecnologia composta de hardware+software que auxilie os fisioterapeutas a monitorar e avaliar de forma objetiva e quantitativa a evolução motora de pacientes com sequelas no tornozelo após um AVC. Projeto elaborado para o Project Based Learning do quarto semestre do Grupo 4 dos alunos de graduação em Engenharia Biomédica da Faculdade Israelita de Ciências da Saúde Albert Einstein (FICSAE).
